
# 二次元签到插件 (Qsign Plus)

二次元图片签到以及好友买卖插件，支持群聊互动玩法。

## 基础功能
- 签到 - 每日签到获得金币
- 签到查询 - 查看个人资产和预计收益
- 购买@用户 - 雇佣其他用户为雇员
- 出售@用户 - 出售自己的雇员
- 赎身 - 雇员支付金币解除雇佣关系
- 存款[空格][金币数量] - 将金币存入银行
- 取款[空格][金币数量] - 从银行取出金币
- 排行榜/财富榜 - 查看群内财富排行

## 股市系统
- 购买 @群主 数量 - 使用金币购买群主货币
- 出售 @群主 数量 - 出售群主货币获得金币
- 汇率 - 查看近7天汇率历史和当前汇率

汇率系统使用几何布朗运动 (GBM) 和均值回归算法模拟波动，支持三种趋势模式：
- **random 模式**: 自动在牛市、熊市、震荡市之间随机切换
- **fixed 模式**: 使用固定的趋势方向（上涨/下跌/震荡）
- **off 模式**: 关闭趋势，仅使用均值回归模型

## 注意事项
- 应该仅支持QQ客户端，且非官方bot
- 所有指令需要@机器人才能触发
- 只有在配置的白名单群聊中才能使用功能

## 玩法说明
- 签到获得金币（根据基础身份、连续签到、雇员数量、银行利息等判定，具体可通过签到查询指令看预计奖励）
- 根据金币（身价）获得身份等级
- 不同身价每天签到基础值不同，不同身价的雇员提供的加成不同
- 被购买后成为雇员，每天签到收益减少（可通过配置调整）

## 数据存储
- 所有用户数据存储在SQLite数据库中
- 数据库路径：`data/plugin_data/astrbot_plugin_Qsign/qsign.db`
- 支持群组隔离，不同群聊的数据互不影响

## 配置项
配置通过AstrBot WebUI进行，分为以下分类：

### 基础配置
- **允许使用功能的QQ群列表** - 只有在列表中的群才能触发插件功能，为空则允许所有群
- **背景图片API地址** - 用于签到卡片背景的随机图片API

### 交易配置
- **恶意收购额外费用率** - 收购他人雇员时需额外支付的费用比例
- **主动出售返还率** - 出售雇员时可收回的身价比率
- **赎身时原雇主得款率** - 雇员赎身时原雇主获得的补偿比例

### 契约配置
- **被雇佣时收入惩罚率** - 雇员签到时的收益惩罚比例
- **每级契约身价加成** - 契约等级每提升1级的身价加成
- **每级契约增益加成** - 契约等级每提升1级的收益加成

### 管理员配置
- **管理员价格加成比例** - 管理员/群主的身价加成比例

### 股市配置
- **基准汇率** - 1群主货币 = X 金币，默认 1.0
- **波动率** - 几何布朗运动波动率参数，默认 0.02
- **均值回归速度** - 均值回归速度参数，默认 0.1
- **均值回归水平** - 长期均衡汇率，默认 1.0
- **更新间隔** - 汇率更新间隔（分钟），默认 60
- **历史天数** - 汇率历史保存天数，默认 7
- **趋势模式** - random/fixed/off，默认 off
- **固定趋势方向** - 1=牛市，-1=熊市，0=震荡
- **牛市概率** - 随机进入牛市的概率（%），默认 30
- **熊市概率** - 随机进入熊市的概率（%），默认 30
- **震荡概率** - 随机进入震荡市的概率（%），默认 40
- **趋势最少持续天数** - 趋势至少持续的天数，默认 3
- **趋势最多持续天数** - 趋势最多持续的天数，默认 10
- **最小趋势强度** - 每天趋势影响的最小幅度，默认 0.01
- **最大趋势强度** - 每天趋势影响的最大幅度，默认 0.05
